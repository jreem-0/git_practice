Welcome

This page demonstrates a fully accessible, CSS-variable-driven dark mode toggle. Your preference is saved in localStorage and restored on every visit. The initial value falls back to your operating-system preference (prefers-color-scheme).

How it works

Toggling the switch adds or removes the dark class on <html>. All colours are defined as CSS custom properties so the entire palette switches in a single re-paint — no JavaScript style manipulation required. See darkmode.js and styles.css for the implementation.
