## Common Tasks

- **Adding files:** `git add <file>` or `git add .` to stage all changes
- **Committing:** `git commit -m "your message"` to commit staged changes
- **Checking status:** `git status` to see current working tree status
- **Viewing history:** `git log` to view commit history
- **Branching:** `git checkout -b <branch-name>` to create and switch to a new branch
- **Pushing:** `git push origin <branch-name>` to push changes to remote
