## Common Tasks

When working in this repository, typical tasks include:

- **Adding files** – `git add <file>`
- **Committing changes** – `git commit -m "your message"`
- **Checking status** – `git status`
- **Viewing history** – `git log --oneline`
- **Branching** – `git checkout -b <branch-name>`
